

### Features

Base features
- User accounts
- User roles
- Granular file permissions
- System (admin) permissions

File features
- Public file sharing with a link and password
- Text file editing
- Video streaming
- File sorting
- Symbolic link functionality can be toggled on
- Basic file searching

UX features
- Easy access to a file's permissions
- Resizable file previews (grid and rows)
- Light and dark UI mode
- Responsive UI on PC and mobile

File tabs
- All files
- Home folder
- Saved files
- Accessible files (list of files a user has permission to access)
- Shared files (list of files a user has shared)

Security features
- Account 2FA
- Option to enforce 2FA for a user
- Files need to be explicitly exposed to be accessible
- Admin UI for logs
